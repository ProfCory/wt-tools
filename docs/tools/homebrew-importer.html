Feed it .json, .txt, .md, or .csv files (or paste text). It pulls out every JSON object it can find — whole files and fenced ```json blocks buried in notes — parses them, checks them against the 5etools homebrew structure (_meta.sources + content arrays), lints the {@tag} syntax, then merges everything into one clean, downloadable homebrew file. 100% in-browser: no upload, no server, works offline on your local game network. PDFs aren't parsed here — extract their text first (that's the reliable path anyway).
