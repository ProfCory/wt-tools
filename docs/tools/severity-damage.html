Severity Damage Expander

Expands the DMG 2024 “Damage Severity by Level” core chart into a draftable stat line.
← W&T Tools

Pick a party level and a threat severity. The tool resolves the level tier, hands you the damage / DC / AC / HP core line, then expands it: place a concrete AC & HP inside the band with the role slider, pick a Standard Action / Biome / Monster Type to build a coherent kit (swap any slot for an alternate), and copy a ready-to-run draft. Numbers and combos trace to the matrix in docs/monster-drafting-matrix.md.
